XCode icon resizer

This little tool will generate a zipped AppIcon.appiconset directory containing all required sizes for iOS or Mac application icons from one 1024x1024 png,jpg,bmp,gif image file, ready to be imported in xcode image asset catalog.

Drop here for iOS icons
Drop here for mac icons
Drag your image file and drop it here
Works on Chrome, probably on other browsers too. Most probably will not work on Internet Explorer.
Icons are deleted hourly
